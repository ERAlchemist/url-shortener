API Project: URL Shortener Microservice

User Story:

  • I can POST a URL to https://aqueous-scrubland-34513.herokuapp.com/api/shorturl/new and I will receive a shortened URL in the JSON response.

    Example : {"original_url":"www.google.com","short_url":zMCccMc2p}

  • If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}

  • When I visit the shortened URL, it will redirect me to my original link.

Short URL Creation

example: POST https://aqueous-scrubland-34513.herokuapp.com/api/shorturl/new/https://www.freecodecamp.com

Example Usage:

https://aqueous-scrubland-34513.herokuapp.com/zMCccMc2p

Will Redirect to:

https://freecodecamp.com

by freeCodeCamp
