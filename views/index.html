API Project: URL Shortener Microservice

User Story:

  • I can POST a URL to [base_url]/api/shorturl/new and I will receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":7C_sxAi5h}

  • If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}

  • When I visit the shortened URL, it will redirect me to my original link.

Short URL Creation

example: POST [base_url]/api/shorturl/new - https://www.freecodecamp.com

Example Usage:

[base_url]/7C_sxAi5h

Will Redirect to:

https://freecodecamp.com
